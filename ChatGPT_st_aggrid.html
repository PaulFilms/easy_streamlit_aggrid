Tú dijiste:
estoy usando el componente aggrid para streamlit, he visto en la documentacion del componente que tiene una barra inferior para indicar el numero de filas totales y filtradas, como puedo integrarlo con streamlit?

ChatGPT Plus

En Ag-Grid (AG Grid) la barra inferior que muestra:

  • número total de filas

  • número de filas filtradas

  • filas seleccionadas

  • agregaciones

se llama Status Bar.

En streamlit-aggrid, debes activarla manualmente en las gridOptions.

✅ Cómo activar la barra inferior (Status Bar)

Debes configurar statusBar dentro de gridOptions.

Ejemplo completo en Streamlit:

python
import streamlit as st
import pandas as pd
from st_aggrid import AgGrid, GridOptionsBuilder

# Datos de ejemplo
df = pd.DataFrame({
    "Nombre": ["Ana", "Luis", "Carlos", "Marta", "Pedro"],
    "Edad": [23, 35, 29, 41, 30],
    "Ciudad": ["Madrid", "Sevilla", "Valencia", "Bilbao", "Barcelona"]
})

# Construir opciones del grid
gb = GridOptionsBuilder.from_dataframe(df)

# Activar filtros
gb.configure_default_column(filter=True, sortable=True)

# 👇 Activar barra de estado
gb.configure_grid_options(
    statusBar={
        "statusPanels": [
            {"statusPanel": "agTotalRowCountComponent", "align": "left"},
            {"statusPanel": "agFilteredRowCountComponent"},
            {"statusPanel": "agSelectedRowCountComponent"},
            {"statusPanel": "agAggregationComponent"}
        ]
    }
)

gridOptions = gb.build()

# Mostrar grid
AgGrid(
    df,
    gridOptions=gridOptions,
    enable_enterprise_modules=True,  # necesario para algunos paneles
    fit_columns_on_grid_load=True
)

Algunos componentes de la barra de estado requieren:

python

Si no lo activas, puede que:

  • No aparezcan ciertos contadores

  • No funcione la agregación

✅ Ejemplo completo en Streamlit

python
import streamlit as st
import pandas as pd
from st_aggrid import AgGrid, GridOptionsBuilder

st.title("Ejemplo Status Bar - AG Grid")

# Datos de ejemplo
df = pd.DataFrame({
    "Producto": ["A", "B", "C", "D", "E"],
    "Categoria": ["Tech", "Tech", "Hogar", "Hogar", "Tech"],
    "Precio": [100, 200, 150, 300, 250],
    "Cantidad": [5, 3, 8, 2, 6]
})

# Construir grid
gb = GridOptionsBuilder.from_dataframe(df)

# Activar filtros, ordenación y selección
gb.configure_default_column(
    filter=True,
    sortable=True,
    resizable=True
)

gb.configure_selection("multiple")

# 🔹 Configuración de Status Bar
gb.configure_grid_options(
    statusBar={
        "statusPanels": [
            # 🟢 GRATIS
            {
                "statusPanel": "agTotalRowCountComponent",
                "align": "left"
            },
            {
                "statusPanel": "agFilteredRowCountComponent"
            },
            {
                "statusPanel": "agSelectedRowCountComponent"
            },
            # 🔵 ENTERPRISE
            {
                "statusPanel": "agAggregationComponent"
            }
        ]
    }
)

gridOptions = gb.build()

AgGrid(
    df,
    gridOptions=gridOptions,
    enable_enterprise_modules=True,  # necesario para aggregation
    fit_columns_on_grid_load=True
)

🚀 Copiar automáticamente la celda seleccionada (botón Streamlit)

Si quieres un botón que copie el valor seleccionado:

python
import streamlit as st
import pandas as pd
from st_aggrid import AgGrid, GridOptionsBuilder, GridUpdateMode

df = pd.DataFrame({
    "Nombre": ["Ana", "Luis", "Carlos"],
    "Edad": [23, 35, 29],
})

gb = GridOptionsBuilder.from_dataframe(df)

# Activar selección de celda
gb.configure_grid_options(
    enableRangeSelection=True,
    enableCellTextSelection=True
)

gridOptions = gb.build()

grid_response = AgGrid(
    df,
    gridOptions=gridOptions,
    update_mode=GridUpdateMode.SELECTION_CHANGED
)

# Obtener celda seleccionada
selected = grid_response.get("selected_rows")

if selected:
    st.write("Fila seleccionada:", selected)

python
import streamlit as st
import pandas as pd
from st_aggrid import AgGrid, GridOptionsBuilder, JsCode

st.title("Simulación Tree (Community)")

# -----------------------------
# Datos base jerárquicos
# -----------------------------
data = [
    {"tipo": "padre", "id": 1, "parent": None, "nombre": "Pedido 1001"},
    {"tipo": "hijo", "id": 2, "parent": 1, "nombre": "Producto A"},
    {"tipo": "hijo", "id": 3, "parent": 1, "nombre": "Producto B"},
    {"tipo": "padre", "id": 4, "parent": None, "nombre": "Pedido 1002"},
    {"tipo": "hijo", "id": 5, "parent": 4, "nombre": "Producto C"},
]

df_base = pd.DataFrame(data)

# -----------------------------
# Estado expandido
# -----------------------------
if "expanded" not in st.session_state:
    st.session_state.expanded = set()

# -----------------------------
# Construir dataframe visible
# -----------------------------
rows = []

for _, row in df_base.iterrows():
    if row["tipo"] == "padre":
        is_expanded = row["id"] in st.session_state.expanded
        icon = "🔽" if is_expanded else "▶"

        rows.append({
            "tree": f"{icon} {row['nombre']}",
            "id": row["id"],
            "tipo": "padre"
        })

        if is_expanded:
            children = df_base[df_base["parent"] == row["id"]]
            for _, child in children.iterrows():
                rows.append({
                    "tree": f"    ↳ {child['nombre']}",
                    "id": child["id"],
                    "tipo": "hijo"
                })

df = pd.DataFrame(rows)

# -----------------------------
# Configurar grid
# -----------------------------
gb = GridOptionsBuilder.from_dataframe(df)

gb.configure_default_column(resizable=True)

# Estilo padres en negrita
cell_style = JsCode("""
function(params) {
    if (params.data.tipo === 'padre') {
        return {fontWeight: 'bold'};
    }
}
""")

gb.configure_column("tree", headerName="Pedidos", cellStyle=cell_style)

gb.configure_selection(
    selection_mode="single",
    use_checkbox=True
)

gb.configure_grid_options(
    rowSelection="single",
    suppressRowClickSelection=True,
    enableRangeSelection=True
)

gridOptions = gb.build()

grid_response = AgGrid(
    df,
    gridOptions=gridOptions,
    allow_unsafe_jscode=True,
    fit_columns_on_grid_load=True
)

# -----------------------------
# Detectar click en padre
# -----------------------------
selected = grid_response["selected_rows"]

if selected:
    selected_row = selected[0]
    if selected_row["tipo"] == "padre":
        parent_id = selected_row["id"]

        if parent_id in st.session_state.expanded:
            st.session_state.expanded.remove(parent_id)
        else:
            st.session_state.expanded.add(parent_id)

        st.rerun()

    sortable=True,

🧠 1️⃣ ¿Por qué no aparece la marca de agua Enterprise?

No hay trampa rara, pero sí contexto técnico:

En AG Grid Enterprise la watermark aparece cuando:

  • Se usa código Enterprise

  • Y la librería detecta que no hay licencia válida

  • Y el build no está “ofuscado” o modificado

En streamlit-aggrid:

  • El wrapper incluye los módulos Enterprise

  • Pero no siempre activa el sistema de licencia visual

  • En entornos locales muchas veces no aparece watermark

  • Puede aparecer solo en ciertos modos o builds

👉 No significa que sea gratis usar Enterprise en producción.
👉 Significa que el wrapper no siempre fuerza la verificación visual.

Si activas cosas como:

  • treeData

  • pivot

  • masterDetail

  • agAggregationComponent

Es más probable que aparezca la marca.

🎯 2️⃣ Cómo renderizar la SideBar cerrada por defecto

Si usas:

python
sideBar=True

Se abre automáticamente.

Para que esté cerrada al inicio, debes definirla como objeto y NO poner defaultToolPanel.

✅ Opción simple: cerrada por defecto

python
gb.configure_grid_options(
    sideBar={
        "toolPanels": [
            {
                "id": "columns",
                "labelDefault": "Columnas",
                "iconKey": "columns",
                "toolPanel": "agColumnsToolPanel"
            },
            {
                "id": "filters",
                "labelDefault": "Filtros",
                "iconKey": "filter",
                "toolPanel": "agFiltersToolPanel"
            }
        ]
        # 👇 NO poner defaultToolPanel
    }
)

Si no defines defaultToolPanel, arranca cerrada.

✅ Opción explícita (forzar cerrada)

Puedes añadir:

python
gb.configure_grid_options(
    sideBar={
        "toolPanels": [...],
        "hiddenByDefault": True
    }
)

Esto fuerza que el panel esté oculto al renderizar.
